# Formula 1 Data Analytics Dashboard 🏎️

This project analyzes historical Formula 1 race data and presents insights using Python and Streamlit.

## Features

- Top Drivers by Wins
- Top Constructors by Wins
- Circuits Hosting the Most Races

## Technologies Used

Python  
Pandas  
Matplotlib  
Streamlit  

## Run the Dashboard

Install dependencies:

pip install -r requirements.txt

Run the dashboard:

python -m streamlit run dashboard/app.py
